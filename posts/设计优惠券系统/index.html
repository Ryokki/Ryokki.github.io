设计优惠券系统

16 May 2024
@Kaijian Xu

目标

设计一个通用的优惠券系统，供以后使用。目前想到的场景如下

  1. 绑定邀请码后自动给用户发放一些优惠券（项目版本、算力包、加油包 各一张，折扣10%，有效期一个月 ）
  2. 后续会把pro版买算力包9折、服务商版算力包7折变成优惠券（这个不搞成优惠券，这个算另一种折扣？）
  3. 后续可能有一些发优惠券的活动

系统设计

优惠券

  1. 优惠券名称
  2. 覆盖范围：商品/组合/行业/全平台
  3. 优惠形式：折扣/固定金额
  4. 使用限制：订单（满足金额）/用户（版本、注册时间超过X）/项目（版本）
  5. 使用次数：1次/多次
  6. 使用时间：固定有效期/推移量

优惠券逻辑主要是两阶段

  1. 发放券阶段：什么时候，发给谁，什么券
  2. 领券后：什么用户，什么时间，什么订单可用优惠券。能优惠多少。

createStripeSubscription

@Transactional
public StripeResult createStripeSubscription(
  List<ProductAndPurchasedDetail> productAndPurchasedDetail, long accountId) {
List<SubscriptionCreateParams.Item> items =
	productAndPurchasedDetail.stream()
		.map(
			detail ->
				SubscriptionCreateParams.Item.builder()
					.setPrice(getStripePriceIdByProductId(detail.getProductId()))
					.build())
		.collect(Collectors.toList());
String customerId =
	stripeCustomerInfoRepository.findCustomerIdByAccountId(accountId).orElseThrow();
SubscriptionCreateParams subCreateParams =
	SubscriptionCreateParams.builder()
		.setCustomer(customerId)
		.addAllItem(items)
		.build();
// create stripe subscription and save 
subscription = Subscription.create(subCreateParams);
StripeSubscription savedStripeSubscription =
	stripeSubscriptionRepository.save(
		new StripeSubscription(
			null,
			subscription.getId(),
			subscription.getCustomer(),
			Utils.convertEpochSecondsToOffsetTime(subscription.getCurrentPeriodStart()), // 这里是创建订阅的时候就定了，current period... alipays是在payment callback中设的.. what fuck..
			Utils.convertEpochSecondsToOffsetTime(subscription.getCurrentPeriodEnd()),
			SubscriptionStatus.valueOf(subscription.getStatus().toUpperCase()),
			subscription.getLatestInvoice(),
			OffsetDateTime.now()));
// Create Order and Payment
Payment payment =
	createOrderAndPaymentByStripeSubscription(
		false, subscription, savedStripeSubscription.getId(), productAndPurchasedDetail);
return new StripeResult(
	HashidsUtil.encode(payment.getId()),
	HashidsUtil.encode(paymentService.getOrderIdByPaymentId(payment.getId())),
	subscription.getLatestInvoiceObject().getPaymentIntentObject().getClientSecret(),
	null);
}

cyclePayAfterSign

// step1: createOrder, createPayment
// step2: createAlipaySubscription, save alpay_subscription, for schedule trigger new payment.
@SneakyThrows
@Transactional
public AlipaySubscriptionResult cyclePayAfterSign(
  long accountId,
  ProductAndPurchasedDetail productAndPurchasedDetail,
  String subjectName,
  PaymentCycle paymentCycle,
  int freeTrialDays) {
OffsetDateTime firsExecuteTime = generateExecuteTime(freeTrialDays);
boolean hasFreeTrial = freeTrialDays != 0;
// 创建订单，如果freeTrial就金额为0
PurchaseOrder order =
	orderService.createOrder(
		accountId, List.of(productAndPurchasedDetail), hasFreeTrial ? BigDecimal.ZERO : null);
String outTradeNo = UUID.randomUUID().toString();
// 创建payment
Payment payment = paymentService.createPayment(order, PaymentType.ALI_PAY, outTradeNo);
String externalAgreementNo = UUID.randomUUID().toString();
AlipaySubscription aliCyclePay =
	createAliCyclePay(
		accountId,
		productAndPurchasedDetail.getProductId(),
		order.getId(),
		externalAgreementNo,
		firsExecuteTime);
AlipaySignScene signScene =
	tryToCreateAccountSignScene(accountId, aliCyclePay.getId()).getSignScene();
createAliPaymentInfo(payment.getId(), aliCyclePay.getId(), outTradeNo);

// 发起支付宝订阅
BigDecimal price = productAndPurchasedDetail.getPurchasedProductDetail().getPrice();
Pair<OffsetDateTime, CyclePayAfterSignBizContent> executeTimeAndCyclePayAfterSignBizContent =
	generateCyclePayAfterSignBizContent(
		signScene, subjectName, price, paymentCycle, externalAgreementNo, firsExecuteTime);
AlipayUserAgreementPageSignRequest request = new AlipayUserAgreementPageSignRequest();
request.setBizContent(
	JsonUtils.toJsonString(executeTimeAndCyclePayAfterSignBizContent.getRight()));
request.setNotifyUrl(notifyUrl);
AlipayUserAgreementPageSignResponse response = alipayClient.pageExecute(request);
return new AlipaySubscriptionResult(
	HashidsUtil.encode(payment.getId()),
	response.getBody(),
	executeTimeAndCyclePayAfterSignBizContent.getLeft());
}
